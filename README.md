# Data Cleaning & Visualization Project

## Overview
This project demonstrates data cleaning, preprocessing, exploratory data analysis (EDA), and visualization using the Superstore dataset.

## Dataset
- Superstore Dataset

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Tasks
- Imported the dataset
- Handled missing values
- Removed duplicate records
- Detected and treated outliers
- Performed data preprocessing
- Conducted Exploratory Data Analysis (EDA)
- Created visualizations
- Generated business insights

## Key Visualizations
- Sales by Category
- Profit by Region
- Monthly Sales Trend
- Top Products
- Segment-wise Sales Analysis

## Project Outcome
The project demonstrates practical skills in data preprocessing, data visualization, and extracting actionable insights from business data.
